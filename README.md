# 💧 PrimeWater Smart CSM
### **Advanced Customer Service Management & Infrastructure Monitoring**

A premium, modern web application designed for **PrimeWater Malaybalay** to bridge the gap between residents and city water administration. This platform provides real-time infrastructure tracking, smart support, and a unified dashboard for all water-related services.

---

## ✨ Features

### 🏢 Admin Command Center
- **Real-Time Map Integration**: Plot and track active water issues across Malaybalay City with precise GPS data.
- **Smart Support Chat**: direct two-way communication channel with residents using a modern, threaded interface.
- **Infrastructure Analytics**: Visual data representing incident resolution rates, water consumption trends, and common service areas.

### 🏠 Resident Portal
- **One-Click Incident Reporting**: Report leaks or water issues instantly with photo and GPS location.
- **Smart Support Messaging**: A modern "Message Center" (formerly Messenger) for fast communication with PrimeWater admins.
- **Billing & History**: Unified view of account credentials, latest water bills, and action history.

### 🧠 Smart Infrastructure
- **AI-Powered Support**: Integrated Python-based reasoning center for automated incident classification and common question handling.
- **PWA (Progressive Web App)**: Installable on any phone or desktop, working even in low-signal environments.

---

## 🛠️ Tech Stack

- **Frontend**: React (Vite) + Vanilla CSS (Premium Glassmorphism + Dark Mode)
- **Database**: Supabase (Real-time DB, Auth, and Storage)
- **Icons**: Lucide-React
- **Maps**: React-Leaflet (OpenStreetMap)
- **AI Backend**: Python (Scikit-learn, joblib)

---

## 🚀 Quick Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Environment Setup**:
   - Create a `.env` file with your `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
3. **Run Development**:
   ```bash
   npm run dev
   ```
4. **Mobile Testing**:
   ```bash
   # Share the app to your phone
   npm run share
   ```
